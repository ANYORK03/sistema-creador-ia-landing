Kit de Inicio IA

3 herramientas. 1 sistema.

Estas son las únicas que necesitas para empezar. No más. No menos.

1
ChatGPT Diseño · Contenido

Lo uso para crear imágenes, diseños visuales y tareas básicas de contenido. Es tu asistente creativo — le describes lo que necesitas y lo genera en segundos.

1 Entra a chat.openai.com y crea una cuenta gratis
2 Usa el modelo GPT-4o — tiene generación de imágenes incluida
3 Primer prompt: "Crea una imagen para mi negocio de [tu rubro] estilo profesional, fondo oscuro"
Abrir ChatGPT →
2
Claude Proyectos · Sistemas

Lo uso para ejecutar proyectos completos, escribir estrategias y construir sistemas. Es el más inteligente para razonar y seguir instrucciones complejas sin perderse.

1 Entra a claude.ai y crea tu cuenta gratis
2 Crea un Proyecto con el contexto de tu negocio — así no repites información cada vez
3 Primer uso: "Ayúdame a crear un plan de contenido para esta semana para mi negocio de [rubro]"
Abrir Claude →
3
Notion Sistema · Organización

Lo uso para organizar todo el sistema. Aquí vive tu plan de contenido, tus clientes, tus proyectos. Es el cerebro central que conecta todo.

1 Entra a notion.so y crea tu cuenta gratis
2 Crea una página llamada "Mi Sistema IA" — ahí metes todo lo que vas construyendo
3 Usa el AI de Notion (botón espaciador) para generar tablas y contenido directo en la página
Abrir Notion →
